# Random Password Generator

A Python application that generates strong, secure, and customizable random passwords through a simple graphical user interface.

## Features

* 🔐 Generate secure random passwords
* 📏 Choose password length
* 🔠 Include uppercase and lowercase letters
* 🔢 Include numbers
* 🔣 Include special characters
* 📋 Copy password directly to the clipboard

## Technologies Used

* Python
* Tkinter
* secrets
* string
* pyperclip

## Installation

Install the required dependency:

```bash
pip install pyperclip
```

> Tkinter is included with most standard Python installations.

## Usage

Run the application:

```bash
python main.py
```

Select the desired password options and generate a secure password instantly.

## Security

This application uses Python's **`secrets`** module, which is designed for generating cryptographically secure random passwords.
