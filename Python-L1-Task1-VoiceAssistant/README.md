# Voice Assistant

A Python-based voice assistant that listens to user commands, converts speech to text, generates intelligent responses, and speaks them back using text-to-speech.

## Features

* 🎤 Voice input through the microphone
* 🗣️ Speech-to-text conversion
* 🤖 AI-powered responses
* 🔊 Text-to-speech output
* 👋 Startup greeting message
* ❌ Voice commands to exit the assistant
* 🔧 Easily extendable with custom commands

## Technologies Used

* Python
* Whisper
* SoundDevice
* NumPy
* SciPy
* Requests
* gTTS
* Pygame
* Ollama

## Installation

Install the required dependencies:

```bash
pip install openai-whisper sounddevice numpy scipy requests gtts pygame
```

## Usage

Run the application:

```bash
python main.py
```

Speak your commands through the microphone, and the assistant will process and respond accordingly.
